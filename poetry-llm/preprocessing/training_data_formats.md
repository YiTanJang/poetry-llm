# 학습 데이터 생성 포맷

## 기본 원칙

같은 시 텍스트를 **여러 관점의 학습 데이터**로 변환하면 모델이 시를 다각도로 이해한다. 단순히 텍스트를 외우는 것이 아니라 시의 **구조, 의도, 효과, 과정**을 종합적으로 배우게 된다. 각 포맷은 모델이 창작자, 평론가, 편집자의 역할을 고루 학습할 수 있도록 유기적으로 설계되었다.

---

## 포맷 1 — 완성시 (기본)

### 개요 및 학습 목적
모델의 기본적인 시상 전개, 행/연 구분, 운율적 발음 처리를 학습하기 위한 가장 기본적인 포맷입니다. 텍스트 내부에 삽입된 특수 토큰(`<시작>`, `<행갈이>`, `<연갈이>`, `<끝>`)과 발음 정보 레이어(예: 음절수 `<음절:N>`, 연음 현상 `<연음:표기>`)를 함께 학습하여, 다음 토큰 예측 과정에서 자연스럽게 시적 호흡과 소리(음악성)를 배치하는 능력을 기릅니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "CompletePoemSchema",
  "type": "object",
  "properties": {
    "text": {
      "type": "string",
      "description": "특수 토큰(<시작>, <행갈이>, <연갈이>, <끝>) 및 발음 정보 레이어가 내재된 시 원문"
    }
  },
  "required": ["text"]
}
```

### 학습 데이터 인스턴스 예시
```json
{
  "text": "<시작>죽는 날까지 하늘을 우러러<음절:14><행갈이>한 점 부끄럼이 없기를<연음:업끼를><음절:10><행갈이>잎새에 이는 바람에도<음절:11><행갈이>나는 괴로워했다<음절:8><연갈이>별을 노래하는 마음으로<음절:11><행갈이>모든 죽어가는 것을 사랑해야지<음절:16><행갈이>그리고 나한테 주어진 길을<음절:13><행갈이>걸어가야겠다<음절:8><연갈이>오늘 밤에도 별이 바람에 스치운다<음절:17><끝>"
}
```

## 포맷 2 — 시 + 시론 페어

### 개요 및 학습 목적
추상적이고 분석적인 미학 평론, 시상 전개 방식, 은유법 등 고차원의 메타 개념과 구체적인 시적 언어 사이의 상관관계를 다각도로 학습시킵니다. 평론적 분석과 시 원문의 쌍을 학습하여 모델이 감정이나 이미지를 구현할 때의 문체적 선택을 심층적으로 연계할 수 있게 만듭니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "PoemPoeticsPairSchema",
  "type": "object",
  "properties": {
    "poem": {
      "type": "string",
      "description": "구조적 특수 토큰이 포함된 시 원문"
    },
    "analysis": {
      "type": "string",
      "description": "해당 시의 주제의식, 수사적 특징, 시적 이미지의 효과 등에 관한 문학적 분석 및 비평"
    }
  },
  "required": ["poem", "analysis"]
}
```

### 학습 데이터 인스턴스 예시
```json
{
  "poem": "<시작>죽는 날까지 하늘을 우러러<행갈이>한 점 부끄럼이 없기를<행갈이>잎새에 이는 바람에도<행갈이>나는 괴로워했다<연갈이>별을 노래하는 마음으로<행갈이>모든 죽어가는 것을 사랑해야지<행갈이>그리고 나한테 주어진 길을<행갈이>걸어가야겠다<연갈이>오늘 밤에도 별이 바람에 스치운다<끝>",
  "analysis": "이 시는 식민지 지식인의 치열한 자아 성찰과 순결성의 의지를 '하늘', '바람', '별'이라는 상징적 자연물을 매개로 선명하게 드러냅니다. 특히 '잎새에 이는 바람'이라는 미세한 촉각적 묘사는 시적 화자가 마주하는 내면의 고뇌와 실존적 흔들림을 보여주며, 마지막 행의 '별이 바람에 스치운다'는 엄혹한 현실 속에서도 운명적 소명을 묵묵히 다하겠다는 단호한 성찰과 극복의 태도를 함축하고 있습니다."
}
```

## 포맷 3 — 창작 노트 → 시 (CoT 파이프라인의 핵심)

### 개요 및 학습 목적
본 프로젝트의 핵심 생성 파이프라인인 Chain of Thought(생각의 사슬)를 모델 내부에 이식하기 위한 포맷입니다. 시를 쓰기 전에 작가의 눈으로 창작 의도, 묘사하고자 하는 감정, 전반적인 시상의 구조화 경로, 이미지 배치 전략을 산문 형태로 서술하고 이를 거쳐 최종 시를 완성하게 유도함으로써, 시적 일관성과 미학적 정교함을 극대화합니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "CreativeNoteToPoemSchema",
  "type": "object",
  "properties": {
    "creative_note": {
      "type": "string",
      "description": "시의 창작 배경, 핵심 감정 상태, 비유 및 묘사 대상, 구조 설계(행갈이와 연 배치 등)에 관한 산문 형태의 시상 설계도"
    },
    "poem": {
      "type": "string",
      "description": "창작 노트의 의도를 바탕으로 논리적이고 예술적으로 전개된 시 텍스트"
    }
  },
  "required": ["creative_note", "poem"]
}
```

### 학습 데이터 인스턴스 예시
```json
{
  "creative_note": "비 내리는 늦은 저녁, 홀로 방 안에 앉아 전등 불빛 아래에서 찾아온 소외감과 내적 고독을 표현하고자 합니다. 창밖에서 들이치는 빗소리가 내 안의 침묵을 더욱 고조시키는 상황을 시각적, 촉각적으로 형상화할 것입니다. 1연에서는 유리창을 두드리는 빗방울과 방 안의 고요한 침묵을 묘사하고, 2연에서는 주황빛 불빛 아래 찻잔을 쥔 시적 화자의 시각적 구도를 부각하겠습니다. 3연에서는 결국 빗소리와 나의 감정이 안팎의 구분 없이 하나로 스며드는 내적 성찰을 담을 것입니다. 호흡의 지연을 주기 위해 명사형이나 조사의 뒤에서 의도적인 행갈이를 수행하도록 유도하겠습니다.",
  "poem": "<시작>창문 틈새로 스며드는 밤의<행갈이>가느다란 빗소리가 귓가를 적실 때<행갈이>방 안은 이미 침묵의 호수가 된다<연갈이>주황색 전등 아래 홀로 쥐어본<행갈이>따뜻한 찻잔의 온기가<행갈이>차가운 손끝을 타고 번져가고<연갈이>창밖의 빗소리는 더는 밖의 것이 아닌<행갈이>내 안에서 소리 없이 흐르는<행갈이>슬픔의 또 다른 이름임을 안다<끝>"
}
```

## 포맷 4 — 수정 과정 데이터

### 개요 및 학습 목적
시의 초기 구상(초고)에서 외부 피드백이나 비평적 성찰을 받아들여 시적 수사를 가다듬고 정제된 시(최종본)로 재창조하는 반복 수정 루프를 학습시킵니다. 감정의 과잉, 상투적인 비유, 리듬의 어색함을 진단한 '비평 의견'을 바탕으로 시의 국소적 혹은 전면적 개작을 조율하는 능력을 극대화하여 인간과 AI 협업 편집 과정의 기초를 다집니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "PoemRevisionSchema",
  "type": "object",
  "properties": {
    "draft": {
      "type": "string",
      "description": "감정 묘사가 직접적이거나 상투적인 단어가 포함된 초기 초고 시"
    },
    "critique": {
      "type": "string",
      "description": "초고의 구상과 이미지의 문제점을 진단하고 개선을 위해 제시된 상세 비평 피드백"
    },
    "revised_poem": {
      "type": "string",
      "description": "비평 피드백을 충실히 반영하여 더욱 정교하고 풍부한 수사로 거듭난 수정된 완성시"
    }
  },
  "required": ["draft", "critique", "revised_poem"]
}
```

### 학습 데이터 인스턴스 예시
```json
{
  "draft": "<시작>차가운 겨울바람이 세차게 분다<행갈이>길가에 서서 눈을 맞으니 눈물이 흐른다<행갈이>너무 슬퍼서 마음이 찢어지는 것 같다<끝>",
  "critique": "'겨울바람', '눈물', '마음이 찢어진다'와 같은 정서적 직설법은 화자의 슬픔을 통속적으로 좁혀버리는 단점이 있습니다. 바람의 감촉이나 눈이 내리는 풍경을 시각적 혹은 입체적인 낯선 이미지(예: 날카로운 촉각, 깊고 어두운 고임)로 탈바꿈하고, 감정을 한 단계 절제하여 간접적인 몸짓 속에서 슬픔이 배어나오도록 개작하십시오.",
  "revised_poem": "<시작>은빛 칼날 같은 바람이 볼을 스치고<행갈이>얼어붙은 가로등 아래 멈춰 선다<행갈이>굳은 주머니 속 웅크린 손끝으로<행갈이>말을 잃어버린 겨울이 고여온다<끝>"
}
```

## 포맷 5 — 시 → 설명 (역방향)

### 개요 및 학습 목적
완성된 시 텍스트를 입력받아 그 안에 내재된 의도, 상징물, 기교, 시상 전개 구조, 특히 행갈이와 연갈이가 미학적으로 미치는 효과를 설명 및 비평하게 하는 역추론 학습 포맷입니다. 시를 다각도로 평가하고 해설할 수 있는 능력을 키움으로써, 모델 스스로가 자신의 출력물을 자체 검증하고 미학적 품질을 높이는 평가자 모델(Critique Model)의 역량을 다집니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "ReverseCommentarySchema",
  "type": "object",
  "properties": {
    "poem": {
      "type": "string",
      "description": "분석 및 해설 대상이 되는 완성시"
    },
    "commentary": {
      "type": "string",
      "description": "시의 수사 기법, 은유 구조, 행갈이/연갈이의 미학적 배치 효과 등을 고찰한 종합적 평론 및 해설"
    }
  },
  "required": ["poem", "commentary"]
}
```

### 학습 데이터 인스턴스 예시
```json
{
  "poem": "<시작>흔들리는 것은<행갈이>바람이 아니다<행갈이>내 마음이다<끝>",
  "commentary": "이 시는 극도의 형식적 긴축을 통해 화자의 실존적 자각을 단 3행으로 집약해 냅니다. 1행 '흔들리는 것은' 이후에 가해진 의도적인 행갈이는 문맥의 술어를 아래로 지연시켜 독자에게 미시적인 호흡의 정지와 긴장감을 유도합니다. 2행의 '바람이 아니다'는 흔히 예측되는 외적 요인을 전면 부정함으로써 시적 관심을 한순간에 좁히며, 3행 '내 마음이다'에서 마침내 모든 동요의 근원지가 화자의 내부에 있음을 밝히며 주객전도의 인식을 완전하게 획득합니다. 자연적 흔들림(바람)을 내재적 깊이(마음)로 승화시키는 데 있어 행갈이 배치가 구조적 중추 역할을 수행하고 있습니다."
}
```

## 포맷 6 — 빈칸 채우기 (구조 학습)

### 개요 및 학습 목적
시의 시각적 형태 및 문체적 리듬감의 골격을 모델에 직접 주입하기 위한 포맷입니다. 시 본문 중 중요한 경계에 해당하는 특수 토큰(`<행갈이>`, `<연갈이>`)이나 특정 시적 시어를 빈칸(`[MASK]`)으로 처리하고 이를 추론하도록 함으로써, 모델이 개별 시어의 선정을 넘어 시의 텍스트가 공간과 호흡 속에서 어떻게 시각적·청각적으로 정지되고 연결되는지에 대한 구조적 문법을 강력히 내재화하게 돕습니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "StructureMaskFillSchema",
  "type": "object",
  "properties": {
    "masked_poem": {
      "type": "string",
      "description": "시 본문 중 일부 구조 토큰이나 중요 어휘가 [MASK]로 대체되어 있는 텍스트"
    },
    "targets": {
      "type": "array",
      "description": "각 MASK에 들어갈 알맞은 정답 리스트",
      "items": {
        "type": "object",
        "properties": {
          "mask_index": {
            "type": "integer",
            "description": "masked_poem 내에서 출현하는 MASK의 0-indexed 기반 위치 번호"
          },
          "correct_token": {
            "type": "string",
            "description": "해당 MASK에 대입되어야 할 정답 특수 토큰 또는 형태소"
          }
        },
        "required": ["mask_index", "correct_token"]
      }
    }
  },
  "required": ["masked_poem", "targets"]
}
```

### 학습 데이터 인스턴스 예시
```json
{
  "masked_poem": "<시작>눈이 오는가 북쪽엔[MASK]함박눈 쏟아져 내리는가[MASK]험한 벌판을 달리는[MASK]울 오빠는 맘 놓고 있느냐<끝>",
  "targets": [
    {
      "mask_index": 0,
      "correct_token": "<행갈이>"
    },
    {
      "mask_index": 1,
      "correct_token": "<연갈이>"
    },
    {
      "mask_index": 2,
      "correct_token": "<행갈이>"
    }
  ]
}
```

## 포맷 7 — 소재 → 시 (창작 요청)

### 개요 및 학습 목적
시적 통제력과 지시 수행(Instruction Following) 능력을 함양시키기 위한 지시어 튜닝(Instruction Tuning)용 포맷입니다. 사용자가 제시하는 특정한 소재, 주제적 프롬프트, 반드시 들어가야 할 필수 시어, 전체적인 감정의 정조(Tone), 지향하고자 하는 구조적 포맷 등의 복합 제약 조건들을 정확하게 수용하면서도, 문학적 품격을 떨어뜨리지 않고 통일성 높은 시적 세계관을 생성해 내는 제어력을 학습합니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "InstructionPoemSchema",
  "type": "object",
  "properties": {
    "prompt": {
      "type": "string",
```

## 포맷 8 — Novelty 실험 데이터

### 개요 및 학습 목적
본 연구 프로젝트의 궁극적인 지향점인 **독창성(Novelty)**을 유도하고 탐색하기 위해 설계된 특수 포맷입니다. 일반적인 코퍼스에서 지극히 드물거나 인접하지 않는 이질적인 두 개념(예: '반도체 공정'과 '전통 시조 율격', '양자역학'과 '상실감의 서정성')을 의도적으로 결합한 입력 콘셉트를 제공하고, 통계적인 시상 분포에서 완전히 비껴난 독창적인 수사 조합을 생성하도록 유도합니다. 평가 도구에 의해 계산된 'Novelty Score'를 데이터셋에 태깅해 학습함으로써, 모델이 일반적인 상투성에 수렴하지 않고 전위적인 어휘적·개념적 도약을 탐구할 수 있는 추진력을 제공합니다.

### 데이터 스키마 (JSON Schema)
```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "NoveltyExperimentSchema",
  "type": "object",
  "properties": {
    "experiment_concept": {
      "type": "object",
      "properties": {
        "subject": {
          "type": "string",
          "description": "기존 서정시나 현대시에서 거의 시도되지 않은 급진적이고 생소한 첨단 과학기술, 기계적 시스템 또는 이종 분야의 소재"
        },
        "style_or_form": {
          "type": "string",
          "description": "해당 기이한 소재를 틀지을 형식적 조건 (예: 고전 시조의 3장 6구 율격, 특정한 외적 압축 형식)"
        }
      },
      "required": ["subject", "style_or_form"]
    },
    "novelty_score": {
      "type": "number",
      "minimum": 0.0,
      "maximum": 1.0,
      "description": "기존 코퍼스와의 시어 임베딩 거리, 바이그램 중복률 등을 종합해 정량화한 미학적 참신성 점수"
    },
    "poem": {
      "type": "string",
      "description": "이질적 개념을 긴밀한 서정성 속에서 융합해 내고 지정된 율격을 충실히 구현한 실험시"
    }
  },
  "required": ["experiment_concept", "novelty_score", "poem"]
}
```

```json

}
```
