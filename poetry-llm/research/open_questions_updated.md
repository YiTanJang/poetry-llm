## 데이터 영역

### Q1. 시론이 시 생성 능력에 실제로 기여하는가?
- **상태**: 🟡 부분 답변 — 강한 간접 근거 확보
- **업데이트**: COIG-Writer (arXiv 2510.14763)가 중국어 창의적 글쓰기에서 "과정 데이터(사고 과정 포함 데이터)"가 없는 경우 체계적 창의성 결핍이 발생함을 보임. 시론은 시적 판단의 언어화된 과정 데이터에 해당한다. 시론 없이 훈련 vs. 시론 포함 훈련 비교 ablation이 없으나, 과정 감독(process supervision)의 효과를 지지하는 근거가 분명히 존재한다.
- **남은 과제**: 한국어 시론의 효과를 한국어 현대시 특화 ablation으로 직접 측정. COIG-Writer의 실험 설계를 참조 가능.
- **우선순위**: ★★★★★ (가장 중요 — 시론 수집의 정당성을 내부적으로 검증해야 함)
- **관련 논문**: [COIG-Writer](https://arxiv.org/abs/2510.14763)

### Q2. 최적 데이터 혼합 비율은?
- **상태**: 🔴 미답 — 실험 필요
- **업데이트**: 합성 데이터 다양성 연구 (arXiv 2511.01490)에서 데이터 품질 필터링이 혼합 비율보다 중요할 수 있음을 시사 ("20% 저품질 포함 100K가 필터된 80K보다 나쁨"). 즉, 혼합 비율 최적화 전에 **각 소스의 품질 기준**을 먼저 정해야 한다.
- **남은 과제**: 현대시 : 시론 : 외국어시 : 보조예술 비율 파일럿 실험. 품질 필터 기준 먼저 설계 필요.
- **우선순위**: ★★★★☆ (파일럿 실험으로 탐색 필요)
- **관련 논문**: [Synthetic Eggs in Many Baskets](https://arxiv.org/abs/2511.01490)

### Q3. 외국어 시가 한국어 시 생성에 어떻게 기여하는가?
- **상태**: 🔴 미답 — 탐색 필요
- **업데이트**: 시 생성 연구에서 다국어 전이가 novelty에 기여한다는 직접 근거는 아직 없다. 영어·중국어·아랍어·포르투갈어·러시아어로 시 생성 연구가 확장되고 있으나 (Evaluating Diversity, arXiv 2406.15267) 언어 간 미학 전이 연구는 별도로 존재하지 않는다.
- **남은 과제**: 일본어 시(하이쿠 포함) 포함 시 한국어 생성 품질 변화 측정. 한자문화권 공유 이미저리의 활성화 패턴 분석.
- **우선순위**: ★★★☆☆ (흥미롭지만 메인 학습 이후 탐색 가능)

## 모델 영역

### Q4. 풀 파인튜닝이 정말 LoRA보다 시 생성에 더 유리한가?
- **상태**: 🔴 미답 — 실험 필요
- **업데이트**: 직접 비교 연구 없음. 단, 고품질 도서 파인튜닝이 전문가 수준 글쓰기를 만들어낸다는 근거 (arXiv 2601.18353)는 도메인 특화 전체 가중치 업데이트의 효과를 간접 지지함. LoRA는 파라미터 효율적이지만 깊은 언어 표현 변화에 한계가 있을 수 있다.
- **남은 과제**: 동일 데이터로 Full FT vs LoRA 비교 실험. 시 특유의 어휘 분포, 행갈이 패턴, 특수 토큰 사용에서 차이가 가장 크게 나타날 것으로 예상.
- **우선순위**: ★★★★☆ (리소스 결정에 직결)
- **관련 논문**: [Can Good Writing Be Generative?](https://arxiv.org/abs/2601.18353)

### Q5. 특수 토큰 초기화 방법이 학습에 영향을 미치는가?
- **상태**: 🔴 미답 — 실험 필요
- **업데이트**: 새로 발견된 직접 근거 없음. `<행갈이>` 토큰을 줄바꿈·구두점 관련 토큰 임베딩의 평균으로 초기화하는 것이 더 빠른 수렴을 낼 것이라는 가설은 유지된다.
- **우선순위**: ★★★☆☆ (학습 초기 비교 실험으로 빠르게 확인 가능)

### Q6. 창작 노트가 별도 학습 단계를 필요로 하는가?
- **상태**: 🟡 부분 답변 — COIG-Writer 설계에서 힌트
- **업데이트**: COIG-Writer는 CoT 데이터를 처음부터 함께 학습하는 방식을 채택하고, 이것이 창의적 글쓰기의 "서사 논리" 측면 학습에 효과적임을 보임. SFT 단계에서 지식과 태도를 순차 학습시키는 방식이 더 유리하다는 대조 가설이 있으나 비교 연구는 없다.
- **남은 과제**: 한국어 시 도메인에서 두 방식 비교. 창작 노트 데이터 비율이 중요할 것으로 예상.
- **우선순위**: ★★★★☆
- **관련 논문**: [COIG-Writer](https://arxiv.org/abs/2510.14763)

## 생성 영역

### Q7. 반복 수정 횟수의 최적값은?
- **상태**: 🔴 미답 — 실험 필요
- **업데이트**: Multi-agent 시 생성 연구 (arXiv 2409.03659)는 에이전트 수와 novelty 사이의 관계를 측정했지만 반복 수정 횟수 최적값 연구는 아직 없음. Plug-and-Play Dramaturge (arXiv 2510.05188)가 서사 대본에서 반복 정제 효과를 보였지만 횟수 최적화는 다루지 않음.
- **남은 과제**: 수정 횟수(1, 2, 3, 5, 10회) 별 인간 평가 + perplexity 추적 실험.
- **우선순위**: ★★★★☆

### Q8. 자기 비평과 외부 비평 모델 중 어느 것이 더 효과적인가?
- **상태**: 🟡 부분 답변 — 외부 비평 우세 근거 확보
- **업데이트**: LLM Review (arXiv 2601.08003)는 블라인드 동료 평가(blind peer review) 구조가 자기 비평보다 창의적 글쓰기 향상에 더 효과적임을 보임. 또한 소규모 모델을 LLM-as-a-Judge (더 큰 모델을 비평자로 활용)로 학습시키는 방식이 창의적 글쓰기 능력 향상에 유효함 (arXiv 2508.21476).
- **업데이트된 가설**: 동일 모델 자기 비평 $<$ 별도 비평 전용 모델 $<$ 더 큰 모델을 심사위원으로 활용.
- **남은 과제**: 한국어 시 도메인에서 세 가지 직접 비교. 비평 언어(한국어 vs 영어)가 품질에 영향을 미치는지도 확인 필요.
- **우선순위**: ★★★★★ (파이프라인 설계 핵심 결정)
- **관련 논문**: [LLM Review](https://arxiv.org/abs/2601.08003), [Igniting Creative Writing](https://arxiv.org/abs/2508.21476)

### Q9. 소재 novelty를 어떻게 유도할 것인가?
- **상태**: 🔴 미답 — 설계 필요
- **업데이트**: Multi-Novelty (arXiv 2502.12700)가 추론 시간(inference-time) 다중 시각 브레인스토밍으로 novelty를 높이는 방식을 제안. 이는 프롬프트 엔지니어링 접근이다. LLM Discussion Framework (arXiv 2405.06373)는 역할극이 창의적 다양성을 높임을 보임.
- **업데이트된 방향**: 학습 데이터 설계(소재 희귀성 주입) + inference-time 브레인스토밍을 병행하는 것이 단일 접근보다 효과적일 가능성 높음.
- **우선순위**: ★★★★☆
- **관련 논문**: [Multi-Novelty](https://arxiv.org/abs/2502.12700)

## 평가 영역

### Q10. AI 생성시가 어떤 기준에서 인간 시와 가장 다른가?
- **상태**: 🔴 미답 (한국어 시 특화) — 일반 근거만 확보
- **업데이트**: 일반 창의적 글쓰기에서 AI 생성물이 전문가 대비 부족한 점: 상상력, 예상치 못한 플롯 전환, 독특한 목소리. 일반 독자는 AI를 선호하는 경향 (arXiv 2601.18353). 한국어 시 특화 Turing test 데이터는 없음.
- **남은 과제**: 한국 현대시 전문가 패널을 활용한 블라인드 평가 실험 설계.
- **우선순위**: ★★★★★ (최종 평가 프레임워크의 기반)

### Q11. Novelty와 미학적 품질의 상관관계는?
- **상태**: 🟡 부분 답변 — 측정 방법론 확보
- **업데이트**: arXiv 2504.09389가 novelty = 원본성 × 품질의 조화 평균으로 측정. 스케일링은 품질을 높이지만 원본성은 안정적 — 두 축이 독립적으로 움직임을 시사. 즉 **novelty와 미학적 품질은 관련은 있지만 별도 축이다.** 전문가는 novelty 높은 시를 선호할 가능성이 있지만, 일반 독자는 coherence 높은 시를 선호할 가능성이 있다 (arXiv 2601.18353 간접 근거).
- **남은 과제**: 한국어 시에서 novelty 점수와 전문가 미학 평가의 상관관계 직접 측정.
- **우선순위**: ★★★★☆
- **관련 논문**: [Measuring LLM Novelty](https://arxiv.org/abs/2504.09389)

## 철학적 영역

### Q12. LLM이 "의도"를 가질 수 있는가?
- **상태**: 🔵 철학적 — 답 없음, 실용적 방향 정리
- **업데이트**: 이 질문은 경험적으로 해결되지 않는다. 실용적 방향: 창작 노트(CoT)를 "의도의 증거"가 아니라 "의도의 기능적 등가물"로 취급한다. 평가 방법론은 의도의 진정성이 아니라 **창작 노트가 최종 시와 미학적으로 일관성이 있는지**를 측정한다.
- **우선순위**: ★★☆☆☆ (실용적으로 우회 가능)

### Q13. 기계가 만든 novelty는 진짜 novelty인가?
- **상태**: 🔵 철학적 — 답 없음, 조작적 정의 채택
- **업데이트**: NoveltyBench (arXiv 2504.05228)가 지적하듯 "originality relative to training data"와 "diversity across generations"는 다른 것이다. 이 프로젝트의 실용적 입장: novelty = "기존 한국 현대시 코퍼스에 없는 미학적 이미지·수사·소재의 조합"으로 조작적으로 정의한다. 이 정의가 인간 시인의 novelty와 동일한지는 답하지 않는다.
- **우선순위**: ★★☆☆☆

## 신규 질문 (2026-06-27 추가)

### Q14. 에코챔버 효과를 평가 단계에서 어떻게 방지할 것인가?
- **상태**: 🔴 미답 — 설계 필요
- **배경**: Tang Poetry 연구 (arXiv 2510.15313)에서 확인. 모델들이 비슷한 데이터로 학습되면 평가 기준도 수렴해 novelty 높은 시를 낮게 평가할 수 있다.
- **질문**: 평가 모델(LLM-as-a-Judge)이 훈련 데이터와 같은 편향을 공유하지 않도록 하려면 어떻게 설계해야 하는가? 평가자를 다른 학습 분포의 모델로 사용해야 하는가?
- **우선순위**: ★★★★☆
- **관련 논문**: [Tang Poetry](https://arxiv.org/abs/2510.15313)

### Q15. CrPO(Creative Preference Optimization)를 시 도메인에 직접 적용할 수 있는가?
- **상태**: 🔴 미답 — 설계 필요
- **배경**: CrPO (arXiv 2505.14442, EMNLP 2025)는 DPO 변형으로 novelty, diversity, surprise, quality를 직접 최적화한다. 창의적 글쓰기 전반을 대상으로 하지만 시 특화 적용은 없다.
- **질문**: CrPO의 선호 쌍(preference pairs)을 한국 현대시에서 어떻게 구성할 것인가? 시의 surprise를 정의할 수 있는가?
- **우선순위**: ★★★★☆
- **관련 논문**: [Creative Preference Optimization](https://arxiv.org/abs/2505.14442)

### Q16. 멀티 에이전트 비평이 단일 모델 반복 수정보다 한국어 시에서 더 유효한가?
- **상태**: 🔴 미답 — 실험 필요
- **배경**: 멀티 에이전트 시 생성이 novelty를 5.6-11.3%p 향상시킨다는 결과 (arXiv 2409.03659). 그러나 한국어 시 생성에서 멀티 에이전트 비교는 없다.
- **질문**: 같은 모델의 자기 수정 N회 vs. 다른 에이전트 역할(화자, 비평자, 편집자) 분리 방식 중 어느 것이 한국어 시에서 더 유효한가?
- **우선순위**: ★★★★★ (파이프라인 설계 핵심)
- **관련 논문**: [Multi-agent Poetry](https://arxiv.org/abs/2409.03659), [LLM Review](https://arxiv.org/abs/2601.08003)

### Q17. 한국어 시에서 인간 전문가의 평가와 일반 독자의 평가는 어떻게 다른가?
- **상태**: 🔴 미답 — 조사 필요
- **배경**: arXiv 2601.18353에서 전문 MFA 작가와 일반 독자가 AI 글쓰기를 다르게 평가함을 확인 (전문가 82.7% 인간 선호 $\rightarrow$ 파인튜닝 후 62% AI 선호). 한국어 시 전문가(시인, 문학평론가)와 일반 독자 사이에도 같은 분열이 있는가?
- **질문**: 이 프로젝트의 평가는 어떤 평가자 집단을 대상으로 설계해야 하는가?
- **우선순위**: ★★★★★ (평가 프레임워크 설계의 전제)
- **관련 논문**: [Can Good Writing Be Generative?](https://arxiv.org/abs/2601.18353)

### Q18. 합성 창작 노트(CoT)의 퇴화를 어떻게 방지할 것인가?
- **상태**: 🔴 미답 — 설계 필요
- **배경**: 모델이 자신의 출력물로 반복 학습되면 다양성이 감소한다 (arXiv 2511.01490). 창작 노트를 LLM으로 합성할 때 이 퇴화가 발생할 수 있다.
- **질문**: 합성 창작 노트의 다양성을 유지하는 방법은? 인간이 작성한 시론 텍스트를 앵커로 사용하는 방식이 효과적인가?
- **우선순위**: ★★★★☆
- **관련 논문**: [Synthetic Eggs](https://arxiv.org/abs/2511.01490)

### Q19. 한국어 형태소 경계와 시적 리듬의 관계를 모델이 학습하는가?
- **상태**: 🔴 미답 — 실험 필요
- **배경**: 한국어 BPE 토크나이저는 형태소 경계를 자주 잘못 절단한다. 시적 리듬은 음절/형태소 단위와 강하게 연결된다.
- **질문**: 형태소 분석기(KoNLPy 등)를 이용한 전처리가 시 생성 품질에 영향을 미치는가? 형태소 경계 인식 특수 토큰이 필요한가?
- **우선순위**: ★★★☆☆ (초기 파이프라인 이후 개선 시도)

### Q20. CreativeBench(arXiv 2603.11863)의 측정 방식이 한국어 시에 적용 가능한가?
- **상태**: 🔴 미답 — 조사 필요
- **배경**: CreativeBench (2026)는 LLM 창의성 벤치마킹 및 향상을 위한 최신 프레임워크. 주로 영어 기반.
- **질문**: CreativeBench의 평가 방식을 한국어 시 도메인에 적용하거나 확장할 수 있는가? 어떤 수정이 필요한가?
- **우선순위**: ★★★☆☆
- **관련 논문**: [CreativeBench](https://arxiv.org/abs/2603.11863)

### Q21. 시적 형식 숙성에 따른 점진적 교육(Curriculum Learning)이 가능한가?
- **상태**: 🔴 미답 — 설계 필요
- **배경**: 복잡한 문학적 형식이나 긴 시를 생성할 때, 처음부터 복잡한 구조를 학습시키면 학습 불안정성이 높아진다.
- **질문**: 초기에 짧은 형태의 단시(2-4행) 및 명확한 구조를 먼저 학습시킨 후, 점차 다층적 이미지와 복잡한 구조의 장시로 확장 학습시키는 커리큘럼 설계가 시적 정밀도 향상에 기여하는가?
- **우선순위**: ★★★★☆

### Q22. 시 음운적 내재율 확보를 위한 발음 레이어(Phoneme-level Embedding) 도입이 유효한가?
- **상태**: 🔴 미답 — 실험 필요
- **배경**: 한국어 텍스트 표기(한글)는 표음문자이지만 실제 발음(연음 법칙, 구개음화 등)과 격차가 있다. 시각적 텍스트 정보만으로는 청각적 리듬(운율)을 모델이 인지하기 어렵다.
- **질문**: 학습 데이터 전처리 단계에서 텍스트를 발음 기호(Phoneme)로 변환해 임베딩 레이어에 병렬 입력하거나, 발음 예측 손실 함수를 추가하는 것이 음운적 조화(청각적 쾌감)를 지닌 시어 생성에 기여하는가?
- **우선순위**: ★★★☆☆

### Q23. 계보별(Lineage-specific) 데이터 세분화 파인튜닝이 시상의 일관성에 영향을 주는가?
- **상태**: 🔴 미답 — 실험 필요
- **배경**: 한국 현대시는 해체시(이상, 황지우 등)와 정통 서정시(서정주, 김소월 등)처럼 극단적으로 상반된 미학 계보를 지닌다. 이를 한 번에 혼합 학습할 경우 스타일이 뒤섞여 상호 붕괴할 수 있다.
- **질문**: 모델에 미학적 계보 태그(예: `[해체시]`, `[서정시]`)를 부여하여 조건부 파인튜닝을 수행할 때 생성 결과물의 미학적 농도가 단일 통합 학습 대비 향상되는가?
- **우선순위**: ★★★★☆

### Q24. 목소리의 일관성 유지를 위해 퓨샷(Few-shot) ICL과 풀 파인튜닝 중 어느 것이 적합한가?
- **상태**: 🔴 미답 — 실험 필요
- **배경**: 시적 화자(Persona)의 고유한 '목소리'나 '말투'는 미세한 문맥 정보에 민감하다.
- **질문**: 추론 단계에서 특정 시인의 대표 시 3편을 프롬프트에 제공하는 Few-shot In-Context Learning 방식과, 해당 스타일로 풀 파인튜닝하는 방식 중 화자의 정체성 유지와 어휘 다양성(Novelty) 보존 면에서 어느 쪽이 우세한가?
- **우선순위**: ★★★★☆

### Q25. 모델 가지치기(Pruning) 및 양자화(Quantization)가 시의 창의적 일탈 어휘 생성에 악영향을 미치는가?
- **상태**: 🔴 미답 — 실험 필요
- **배경**: 대형 모델을 실제 서비스에 배포할 때 경량화(FP8, INT4 등)는 필수적이다. 그러나 경량화는 확률 분포의 미세한 꼬리(Long-tail) 부분을 절단하여 창의적 단어 선택을 억제한다는 연구 보고가 있다.
- **질문**: 32B FP16 모델을 INT4 또는 INT8 수준으로 양자화했을 때, 시 생성의 Distinct-k 다양성 점수 및 시적 조어(Neologism) 발생 비율의 감소폭은 어느 정도인가?
- **우선순위**: ★★★☆☆
