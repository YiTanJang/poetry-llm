# 기존 AI 시 생성 시스템의 실패 패턴 분석

이 문서는 현존 AI 시 생성 접근법들의 구조적 실패를 기계적으로 분석하고,
이 프로젝트가 각 실패에 어떻게 대응하는지, 그리고 피할 수 없는 근본 한계가 무엇인지를 정리한다.

---

## 1. 범용 LLM의 시 — 왜 "통계적 평균"인가

### 1.1 메커니즘: 가장 가능성 높은 다음 토큰의 축적

LLM의 시 생성은 다음 토큰의 조건부 확률 `P(w_t | w_1, ..., w_{t-1})`을 반복 샘플링한다.
훈련 목표(cross-entropy loss)는 학습 코퍼스에서 자주 공기(共起)하는 패턴을 재현하도록 최적화된다.
시 코퍼스에서 "달"과 공기하는 단어는 "그리움", "외로움", "하얀"이 압도적이다.
따라서 모델이 달을 쓸 때 가장 높은 확률로 선택하는 후속 어휘는 이 클리셰 집합에 집중된다.

**증거**: ChatGPT의 시 생성이 "초인적 평범함(superhuman banality)"을 보인다는 연구 결과 (Antislop, arXiv 2510.15061).
Greedy decoding은 자기 강화(self-reinforcement) 효과로 반복이 점점 더 강화된다.
반복이 일어날수록 반복 토큰의 평균 확률이 상승하는 경향이 확인됨.

### 1.2 스케일의 역설

모델을 크게 키우면 novelty의 품질(quality) 차원은 향상되지만 **원본성(originality)은 안정적으로 유지** — 즉, 스케일로 originality 문제를 해결할 수 없다 (arXiv 2504.09389).
더 큰 모델은 더 유창하지만 더 클리셰적일 수 있다.

### 1.3 에코챔버 효과

같은 시 코퍼스로 학습한 모델들은 서로 비슷한 시적 품질 기준으로 수렴한다.
고전 한시(당시) 생성 연구에서 확인 (arXiv 2510.15313):
"모델들이 공유하면서도 부정확한 품질 기준을 형성"하며,
**자동 평가 메트릭이 이 문제를 탐지하지 못한다.**
인간 평가자도 같은 코퍼스에 노출된 경우 같은 편향을 공유한다.

### 1.4 요약: 범용 LLM 시의 구조적 실패

| 실패 유형 | 원인 | 증상 |
|----------|------|------|
| 클리셰 | 공기 빈도 극대화 | "하얀 달", "그리운 마음", "파란 하늘" |
| 예측 가능한 리듬 | 운율 패턴 통계적 학습 | 4·4조, 3·4조 반복 |
| 감정 일반화 | 감정 어휘 빈도 편향 | 슬픔/그리움/외로움 과다 |
| 소재 편향 | 시 코퍼스 소재 분포 | 자연/계절/사랑 과집중 |
| 문화 편향 | 영어 시 코퍼스 압도 | 서양 이미저리 침투 |

## 2. 기존 시 특화 파인튜닝 시스템의 공통 실패 원인

### 2.1 형식 제약 없는 파인튜닝: 리듬·운율 붕괴

GPT-2를 시집으로 단순 파인튜닝하면 생성 품질이 일부 향상되지만
**리듬·운율 같은 시 특수 특성을 학습하기 어렵다** (Popescu-Belis et al., 2022).
모델이 의미론적 패턴은 학습하지만 음운론적 패턴은 별도 처리 없이 획득하지 못한다.

**대응 시도**: ByGPT5 (Belouadi & Eger, 2023)는 토큰 없는(token-free) 구조로 리듬·각운·두운을
end-to-end 학습. Hafez (Ghazvininejad et al., 2016)는 형식 제약을 하드코딩.
그러나 이 접근들은 **형식을 제약으로 외부에서 주입**하는 방식이라
모델이 형식을 내면화해서 스스로 선택하는 능력을 키우지 못한다.

### 2.2 소규모 데이터 과적합

시 코퍼스는 일반 텍스트보다 수천 배 작다. 파인튜닝 후 모델이 학습 데이터의
특정 이미지·어구를 거의 그대로 재현하는 memorization 현상이 발생한다.
이는 novelty의 반대 방향이다.

### 2.3 평가 순환 문제

학습 데이터와 같은 분포로 생성된 시를 같은 분포로 평가하면 높은 점수가 나온다.
BLEU·ROUGE 같은 자동 메트릭은 **새로운 것을 낮게 평가**하도록 설계되어 있다.
결과적으로 시스템이 novelty를 잃는 방향으로 최적화된다.

**에코챔버 효과의 평가 버전**: 평가자(자동·인간 모두)가 훈련 코퍼스와 같은 미학적 기준을 공유하면
novelty 높은 시가 불리하게 평가된다 (arXiv 2510.15313).

### 2.4 과정(process) 없는 결과(outcome) 학습

기존 시스템들은 완성된 시 텍스트만으로 학습한다.
**어떤 사유 과정을 거쳐 그 이미지를 선택했는지에 대한 데이터가 없다.**
COIG-Writer (arXiv 2510.14763)가 밝혀낸 것처럼,
창의적 글쓰기는 "서사 논리(과정)" + "언어 표현(결과)"의 이중 구조이며
결과 데이터만으로는 전자를 학습할 수 없다.

### 2.5 단일 생성으로 끝나는 파이프라인

대부분의 시 생성 시스템이 시를 한 번 생성하고 종료한다.
인간 시인은 수십 번의 고쳐쓰기를 거친다. 반복 수정 없는 생성은
초고(draft) 수준의 결과물로 귀결된다.

## 3. Novelty vs. Coherence 트레이드오프 문제

### 3.1 트레이드오프의 구조

| 방향 | 효과 | 위험 |
|------|------|------|
| 높은 온도(temperature) / top-p 확장 | novelty↑ | 문법 붕괴, 의미 불명확 |
| 낮은 온도 | coherence↑ | 클리셰↑, 예측 가능성↑ |
| 훈련 데이터 풍부 | coherence↑ | memorization↑ |
| 소재 주입·제약 | novelty↑ | 억지스러운 이미지 조합 |

### 3.2 기존 연구의 측정 불일치

NoveltyBench (arXiv 2504.05228)는 novelty를 "여러 번 생성 시 서로 다른 답을 낼 수 있는 능력"으로 측정.
arXiv 2504.09389는 novelty를 "원본성 × 품질의 조화 평균"으로 측정.
두 측정이 다른 것을 측정한다: 전자는 inter-generation 다양성, 후자는 학습 데이터 대비 원본성.
**현재 분야에 통일된 novelty 정의가 없다.**

### 3.3 인간 평가와의 불일치

전문 작가(MFA)들은 AI 글쓰기를 선호할 때 "심미적 정체성 위기"를 경험한다 (arXiv 2601.18353).
일반 독자는 AI 글쓰기를 선호하는 경향이 있다.
**novelty가 높을수록 전문가는 높이 평가하고 일반 독자는 낯설게 느낄 수 있다.**
이는 평가 기준 자체가 대상 독자에 따라 달라진다는 것을 의미한다.

## 4. 한국어 시 생성의 특수한 어려움

### 4.1 언어적 어려움

**교착어(Agglutinative) 구조**:
한국어는 어미·조사의 조합이 극도로 다양하다.
"가다"는 "가고", "가서", "가면", "갔는데", "갔더니" 등 수십 가지 변형을 가진다.
시에서 어미 선택은 리듬과 의미 모두에 영향을 미친다.
영어 토크나이저 설계로 만들어진 BPE/SentencePiece 어휘는 한국어 형태소 경계를
자주 잘못 절단해 시의 리듬 학습을 방해한다.

**행갈이(Line Break)의 의미 기능**:
한국 현대시에서 행갈이는 단순한 시각적 배치가 아니라 의미론적 기능을 한다.
침묵, 여운, 전환을 표현한다. 이를 토큰으로 표현하려면 특수 처리가 필요하다 (`<행갈이>` 토큰 등).

**운율 체계의 상이함**:
영어 시는 강세(stress) 기반 운율. 한국어 시는 음절수·어감·음성 상징 기반.
영어 시 운율 연구를 한국어에 직접 이식할 수 없다.

**한자어·순우리말·외래어의 공존**:
현대 한국어는 한자 어원 단어(60% 이상), 순우리말, 영어 외래어가 공존한다.
시인이 "달" 대신 "月" 또는 "moon"을 선택할 때 의미 차이가 발생한다.
LLM이 이 층위 차이를 학습하기 위해서는 층위 정보가 명시된 데이터가 필요하다.

### 4.2 문화적 어려움

**문화 코드의 특수성**:
"오동나무 아래", "두부 장수 소리", "넝마 주이" 같은 이미지는
한국 생활사를 모르면 이해·생성할 수 없다.
범용 LLM의 학습 데이터에서 이런 문화 코드는 희박하다.

**현대 한국시의 미학 계보**:
한국 현대시는 일제강점기 → 전후 → 민중시 → 1980-90년대 해체시 → 2000년대 이후 신서정
의 계보를 가진다. 각 시기의 미학 문법이 다르다.
범용 LLM은 이 계보를 학습할 데이터가 없다.

**시론(詩論)의 중요성**:
한국 시인들이 자신의 미학을 설명한 시론(산문)은
단순히 시를 설명하는 텍스트가 아니라 **미학적 판단 기준의 언어화**다.
이것을 학습하지 않은 모델은 미학적 판단을 할 수 없다.

**번역 편향**:
한국 현대시 중 영어로 번역된 것은 극히 일부이며 번역 과정에서 미학이 손실된다.
범용 LLM이 가진 한국시 지식은 번역본 기반의 편향된 표본이다.

### 4.3 데이터 공백

한국어 시 특화 NLP 데이터셋이 사실상 없다.
KorLit 같은 한국문학 코퍼스는 접근이 제한된다.
공개된 학습 데이터로는 충분한 시 특화 학습이 불가능하다.
→ **이 프로젝트의 직접적 동기**: 시집 ~2,500권 수집을 통한 자체 코퍼스 구축.

## 5. 이 프로젝트의 각 실패에 대한 대응 전략

| 실패 원인 | 이 프로젝트의 대응 | 신뢰도 |
|----------|-----------------|--------|
| 통계적 평균/클리셰 | novelty 기준 데이터 필터링 + 소재 다양성 주입 | 중간 — 검증 필요 |
| 리듬·형식 붕괴 | `<행갈이>` 특수 토큰 + 형식 인식 학습 | 높음 |
| 소규모 데이터 과적합 | 대규모 코퍼스 (~2,500권) + 시론 포함 | 높음 |
| 평가 순환 | novelty 측정 독립 프레임워크 (unseen n-gram 등) | 중간 |
| 과정 없는 학습 | 창작 노트(CoT) — 사고 과정 명시적 학습 | 높음 (COIG-Writer 선례) |
| 단일 생성 | 반복 수정 파이프라인 (다단계 draft → critique → revise) | 중간 |
| 한국어 문화 코드 | 시론 + 시집 + 한국 문화 보조 데이터 | 중간 — 범위 한정 필요 |
| 에코챔버 평가 | 인간 평가 + 자동 평가 병행, 코퍼스 비교 novelty 측정 | 중간 |

## 6. 이 프로젝트도 피할 수 없는 근본적 한계

### 6.1 학습 데이터의 상한

모델은 학습한 텍스트가 표현하는 개념 공간을 벗어날 수 없다.
시집 2,500권은 방대하지만 미래 시인의 미학을 포함하지 않는다.
**진정한 의미의 "기존에 없는 미학"은 학습 데이터 분포 바깥에 있다.**
이 한계는 어떤 방법론으로도 완전히 극복할 수 없다.

### 6.2 novelty 측정의 자기 참조 문제

novelty를 "기존 코퍼스에 없는 n-gram/이미지 조합"으로 측정하면,
기존 코퍼스에서 '드문' 것이 반드시 '미학적으로 새로운' 것은 아니다.
비문법적 오류나 무의미한 조합도 기존 코퍼스에 없다.
**novelty의 조작적 정의와 미학적 정의 사이의 간격은 메울 수 없다.**

### 6.3 평가자 편향의 불가피성

인간 평가자도 훈련받은 미학 기준을 가진다.
한국 현대시 전문가는 특정 미학 계보에 속한다.
"새로운 시"를 평가하는 사람이 "새로운 것"을 인식하는 능력 자체가 기존 미학에 의존한다.
평가자 편향은 평가 설계로 줄일 수 있지만 제거할 수 없다.

### 6.4 합성 데이터의 퇴화(Degeneration)

창작 노트(CoT)를 LLM으로 합성할 때, 모델이 자신의 출력물로 반복 학습되면
**어휘·문법 다양성이 점진적으로 감소**한다 (합성 데이터 다양성 연구, arXiv 2511.01490).
인간 시 텍스트를 앵커로 유지하더라도 합성 사고 과정은 균질화될 수 있다.

### 6.5 "의도"의 시뮬레이션 vs. 실제

창작 노트는 모델이 "의도를 가진 척"하는 텍스트다.
이것이 인간 시인의 의도와 구조적으로 동일한지는 알 수 없다.
평가 방법론이 이 불확실성을 해소하지 못한다면,
우리는 "좋아 보이는 의도 기록"을 최적화할 뿐이다.

### 6.6 한국어 토크나이저 구조적 한계

현재 대형 한국어 지원 LLM(예: EXAONE, HyperCLOVA 계열)의 BPE 토크나이저는
한국어 형태소 경계를 기계적으로 처리한다.
시적 리듬은 형태소 경계와 강하게 연결되어 있는데,
토크나이저가 이 경계를 잘못 절단하면 모델이 리듬을 학습하기 어렵다.
이를 해결하려면 한국어 형태소 분석기 연동 토크나이저가 필요하지만
기존 대형 모델의 토크나이저를 교체하는 것은 현실적으로 불가능에 가깝다.
