# 미답 연구 질문

이 파일은 프로젝트에서 아직 결정되지 않았거나,
더 깊은 탐색이 필요한 질문들을 모은다.
에이전트가 이 질문에 답하는 개념 파일을 만들고 여기서 링크를 연결한다.

---

## 데이터 영역

### Q1. 시론이 시 생성 능력에 실제로 기여하는가?
시론 없이 훈련한 모델 vs 시론 포함 훈련 모델 비교 실험 필요.
시론의 효과를 ablation으로 측정할 수 있는가?

### Q2. 최적 데이터 혼합 비율은?
현대시 : 시론 : 외국어시 : 보조예술 = ?
파일럿 실험으로 탐색 필요.

### Q3. 외국어 시가 한국어 시 생성에 어떻게 기여하는가?
다국어 전이학습(transfer learning)의 창의성 전이 가능성.
일본어 시가 특히 유효할 것이라는 가설 (한자문화권 공유, 여백의 미).

---

## 모델 영역

### Q4. 풀 파인튜닝이 정말 LoRA보다 시 생성에 더 유리한가?
이론적으로는 풀 파인튜닝이 깊은 도메인 적응에 유리하지만
실제로 시 도메인에서 차이가 유의미한지 검증 필요.

### Q5. 특수 토큰 초기화 방법이 학습에 영향을 미치는가?
`<행갈이>` 토큰을 랜덤 초기화 vs 관련 토큰(줄바꿈, 구두점) 평균으로 초기화할 때 차이.

### Q6. 창작 노트가 별도 학습 단계를 필요로 하는가?
CoT 데이터를 처음부터 같이 학습 vs 별도 CoT 파인튜닝 단계 추가 중 어느 것이 유효한가?

---

## 생성 영역

### Q7. 반복 수정 횟수의 최적값은?
너무 적으면 개선 부족, 너무 많으면 과적합·반복에 빠짐.
실험으로 탐색 필요 (목표: perplexity + 인간 선호도 동시 추적).

### Q8. 자기 비평과 외부 비평 모델 중 어느 것이 더 효과적인가?
동일 모델이 생성+비평 vs 별도 비평 전용 모델 혹은 더 큰 모델을 심사위원으로 활용.

### Q9. 소재 novelty를 어떻게 유도할 것인가?
"시에서 잘 다루지 않는 소재" 목록을 학습 데이터 생성 시 의도적으로 주입하는 방법.
프롬프트 엔지니어링으로 가능한가, 학습 데이터 설계가 필요한가.

---

## 평가 영역

### Q10. AI 생성시가 어떤 기준에서 인간 시와 가장 다른가?
Turing test 결과에서 평가자들이 가장 많이 지적하는 차이점.
이를 개선 방향으로 피드백할 수 있는가.

### Q11. Novelty와 미학적 품질의 상관관계는?
novelty가 높은 시가 인간 평가에서도 높은 점수를 받는가,
아니면 이 둘은 서로 다른 축인가.

## 철학적 영역

### Q12. LLM이 "의도"를 가질 수 있는가?
창작 노트(CoT)는 의도의 시뮬레이션인가, 진짜 의도인가.
이 질문이 평가 방법론에 어떤 영향을 미치는가.

### Q13. 기계가 만든 novelty는 진짜 novelty인가?
기존 학습 데이터의 재조합으로 만들어진 새로운 이미지가
인간 시인의 novelty와 질적으로 다른가, 같은가.
