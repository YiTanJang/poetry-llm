# 저작권 만료 한국 시

## 개요

한국 저작권법상 저작권 보호 기간은 저작자 사망 후 70년이다.
이 기간이 경과한 작품은 공공 도메인으로, 별도 허락 없이 사용 가능하다.

## 활용 가능 범위

2026년 기준으로 1956년 이전 사망 시인의 작품이 해당된다.

| 시인 | 생몰 | 주요 특징 |
|------|------|----------|
| 김소월 (1902~1934) | 1934년 사망 → 2004년 만료 | 민요적 리듬, 한의 정서 |
| 이상 (1910~1937) | 1937년 사망 → 2007년 만료 | 다다이즘, 수학적 언어 실험 |
| 이육사 (1904~1944) | 1944년 사망 → 2014년 만료 | 저항시, 남성적 서정 |
| 윤동주 (1917~1945) | 1945년 사망 → 2015년 만료 | 부끄러움, 자기 성찰 |
| 한용운 (1879~1944) | 1944년 사망 → 2014년 만료 | 불교 사상, 님의 침묵 |
| 정지용 (1902~1950) | 1950년 사망 → 2020년 만료 | 이미지즘, 감각적 언어 |

## 오픈 데이터 소스

- **한국문화원연합회** 디지털 자료
- **국립중앙도서관** 고전 자료 디지털화 컬렉션
- **위키문헌(Wikisource)** 한국어 섹션 — 공공도메인 시 수록
- **한국학중앙연구원** 한국향토문화전자대전

## 주의 사항

- 저작권 만료 여부는 **사망 연도** 기준, 출판 연도 기준이 아님
- 시집 편집자의 편집 저작권은 별도 존재 가능
- 번역본의 경우 번역자의 저작권이 별도 발생

## 학습 활용 관점

공공 도메인 시는 저작권 제약 없이 **다양한 형태로 변환 가능**:
- 원본 텍스트
- 구조 분석 버전 (행/연 레이블링)
- 발음 토큰 첨부 버전
- 시론과 페어링한 버전
