# 목표 및 비목표

## 목표 (Goals)

### 1. 모델 목표
- 20~40B 파라미터 모델을 **풀 파인튜닝**하여 시 전용 언어 감각 확보
- 한국어 현대시의 형식적 특징(행갈이, 연갈이, 리듬, 호흡)을 내재화
- 창작 과정을 CoT로 외화(externalize)하여 생성 결과의 추적 가능성 확보

### 2. 데이터 목표
- 한국어 시: 현대시집 ~2,000권, 신춘문예, 메이저 문예지 수록 작품
- 시론/시평론: 가능한 최대한 — 시인이 시에 대해 쓴 글이 가장 중요한 메타데이터
- 외국어 시: 오픈 데이터 + 스캔 ~500권 (영어, 일본어, 중국어, 독일어 등)
- 보조 예술 데이터: 미술, 소설, 극 — 교차 미학 감각 학습용

### 3. 생성 목표
- 단순 출력이 아닌 **반복적 수정 과정**을 통한 시 완성
- 창작 노트(CoT)가 시 텍스트와 함께 진화하는 파이프라인
- Novelty 필터: 기존 코퍼스와 충분히 구별되는 작품만 최종 산출

## 비목표 (Non-Goals)

- **기존 시의 스타일 모방**: 특정 시인의 스타일을 재현하는 것이 목적이 아님
- **대량 생성**: 많은 양의 평범한 시보다 적은 수의 뛰어난 시가 목표
- **영미 중심 시학**: 서구 형식시(소네트, 빌라넬 등) 재현이 목적이 아님
- **즉시 배포 가능한 서비스**: 연구 및 실험이 우선, 서비스화는 이후 고려
- **RLHF 기반 "좋아요" 최적화**: 대중적 선호가 아닌 미학적 기준을 기준으로 삼음

## 성공 기준

| 기준 | 측정 방법 |
|------|----------|
| 행갈이/연갈이 논리 습득 | 구조 정확도 평가 |
| Novelty | 코퍼스 대비 n-gram 유사도, 임베딩 거리 |
| 미학적 평가 | 시인/문학 연구자 블라인드 평가 |
| 창작 노트 일관성 | CoT와 최종 시의 개념적 연결 평가 |
